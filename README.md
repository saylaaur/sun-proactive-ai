# ☀️ Sun Proactive — Autonomous AI Infrastructure
**Project for AIS Hack 3.0**

## 💡 О проекте
**Sun Proactive** — это интеллектуальная экосистема для автоматизации работы фонда «Sun». Система заменяет ручной труд кураторов на 4 автономных AI-агента, которые управляют задачами от создания до проверки результата.

## 🚀 AI Hardcore Features
* **AI Interviewer:** Извлечение Structured Outputs из текста куратора через Gemini 2.5 Flash.
* **Semantic Matching:** Поиск волонтеров через векторные эмбеддинги и Cosine Similarity.
* **Computer Vision Audit:** Мультимодальная верификация выполненных задач по фотоотчетам.
* **Autonomous Agent:** Проактивное управление дедлайнами и уведомлениями.

## 🛠 Tech Stack
* **AI Core:** Gemini 2.5 Flash (Text, Vision, Embeddings).
* **Backend:** FastAPI (Python), Supabase (PostgreSQL + pgvector).
* **Frontend:** Next.js / HTML5 с концепцией Trust UI.
